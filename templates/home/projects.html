Gala Planner (in progress)

Solo project, GitHub
This web app will allow a gala planner to set up tables for sponsors, and the sponsors can invite their guests through this app. Each table guest, including the sponsor, can then provide the planner with all of their information through the web app alone. It will utilize automatic e-mail notifications for easy communication. This idea is based on a suggestion provided by a friend, who happens to be an event planner.
While a few features are still missing, the app can set up galas, tables, and invite new sponsors. In the near future, it will allow sponsors to send guest invitations and set seat details.
Tools used (so far):
- Python 3.6.0
- Django 1.10.5
